# public/js/

⚠️ 一旦所属文件夹有所变化，请更新此文档

## 架构说明

前端JavaScript模块目录，采用模块化设计，支持PO18和POPO双平台。核心模块处理应用逻辑和API通信，功能模块负责具体页面交互（包括书评系统），工具模块提供通用功能和性能优化。

## 文件列表

### 核心模块
- **app.js**: 主应用模块，应用入口和状态管理中心，包含书评系统、书单管理等核心功能
- **api.js**: API通信模块，封装所有后端API调用，支持元信息、书评、章节等新接口
- **utils.js**: 工具函数库，提供防抖节流、懒加载、Toast通知、本地存储等通用功能
- **main.js**: 应用启动文件，初始化应用并启动各个模块
- **module-loader.js**: 模块加载器，负责动态加载和管理各个功能模块

### 功能模块
- **reader.js**: 阅读器模块，实现在线阅读功能，支持阅读设置和进度管理，内置6个主题、颜色自定义、背景图片、字体选择
- **bookshelf.js**: 书架模块，管理用户书架和阅读记录
- **book-detail.js**: 书籍详情模块，展示小说详细信息、章节列表和书评
- **admin.js**: 管理后台模块，提供用户管理、系统监控、书评审核等管理功能
- **rankings.js**: 排行榜模块，展示各类小说排行榜数据
- **i18n.js**: 国际化模块，实现多语言切换功能
- **generator.js**: 文件生成模块，处理TXT/HTML/EPUB格式生成

### 移动端和性能
- **mobile.js**: 移动端适配模块，优化移动设备交互体验
- **mobile-enhancements.js**: 移动端增强模块，提供手势操作和响应式优化
- **app-enhancements.js**: 应用增强模块，提供额外的UI交互和功能增强
- **final-enhancements.js**: 最终增强模块，最后的功能完善和用户体验优化
- **performance-optimizer.js**: 性能优化模块，实现虚拟滚动、资源预加载等性能优化
