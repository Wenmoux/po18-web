# 预设字体目录

此目录用于存放预设字体文件，这些字体会自动加载到阅读器中供用户选择。

## 支持的字体格式

- `.ttf` - TrueType 字体
- `.otf` - OpenType 字体
- `.woff` - Web Open Font Format
- `.woff2` - Web Open Font Format 2.0

## 使用方法

1. 将字体文件放入此目录（`/data/fonts`）
2. 字体文件名将作为字体显示名称（不含扩展名）
3. 重启服务器后，字体将自动出现在阅读器的字体选择列表中

## 示例

如果放入文件 `思源黑体.ttf`，阅读器中会显示为 "思源黑体" 字体选项。

## 注意事项

- 字体文件名建议使用中文或英文，避免特殊字符
- 字体文件大小建议控制在 5MB 以内
- 字体文件会自动加载，无需额外配置
