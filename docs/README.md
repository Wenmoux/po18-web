## 📝 文档说明

### 文档分类原则
1. **项目概览** - 项目整体介绍和历史
2. **功能模块** - 按功能模块分类（游戏、阅读器、主题等）
3. **技术文档** - 性能优化、API、安全等技术细节
4. **开发规划** - 待办事项、功能建议、优化计划
5. **测试文档** - 测试相关文档

### 文档更新规范
- 重要功能更新后，及时更新对应文档
- 新增功能时，创建对应的功能文档
- 定期整理和合并重复内容
- 保持文档结构清晰，便于查找

- 主索引文件：`README.md`
- 文档目录：`docs/`
