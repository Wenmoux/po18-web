# docs/

## 架构说明

项目文档目录，存储项目开发过程中的各类文档，包括项目历史、性能优化、产品规划、待办事项等。用于记录项目演进和指导后续开发。

## 文件列表

- **PROJECT_HISTORY.md**: 项目历史文档，记录项目的开发历程、重要更新和版本变化
- **PERFORMANCE_OPTIMIZATION.md**: 性能优化文档，记录已实施的性能优化措施和效果分析
- **PRODUCT_OPTIMIZATION_TODO.md**: 产品优化待办文档，列出产品功能和用户体验的优化计划
- **todo.md**: 待办事项文档，记录短期待完成的开发任务
- **new.md**: 新功能规划文档，详细记录新功能的设计和实现方案
- **THEME_CUSTOMIZATION.md**: 阅读器主题自定义功能说明，包含功能概述、实现细节、扩展指南（237行）
- **THEME_INTEGRATION_GUIDE.md**: 主题功能集成指南，提供快速开始教程、常见问题和最佳实践（238行）
- **THEME_FEATURE_SUMMARY.md**: 主题功能实现总结，详细记录所有已完成的功能、技术细节和代码统计（369行）
